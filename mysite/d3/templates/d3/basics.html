Similar to jQuery, d3 uses the DOM to select and traverse.

d3.select("p").style("background-color", "red");

Yet styles, attributes, and other properties can be specified as functions of data in D3. For example, the d3.geoPath function projects geographic coordinates into Scalable Vector Graphic (SVG) path data.
